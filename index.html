Instructions

▼

How to build a circuit:

1. Drag components to the canvas

2. Click "Wire Mode" to connect components

3. Click connection points (orange dots) to create wires

4. Click to select components

5. Drag to move components

6. Use properties panel to edit values

7. Shift+Click to rotate components

8. Press Delete to remove selected component

Advanced Mode Features:

• Transistors for amplification and switching

• Diodes for one-way current flow

• Potentiometers for variable resistance

• Voltmeters and Ammeters for measurements

• Logic gates for digital circuits

• Complex circuit analysis with voltage/current calculations

Tips for better circuits:

• Always connect components with wires

• Start with a battery as power source

• Use resistors to limit current for LEDs

• Close switches to complete circuits

• Rotate components for better wiring

• Test frequently with the Simulate button

Troubleshooting:

• No power? Check battery connections

• LED not lighting? Add a resistor

• Circuit not working? Verify all wires are connected

• Components not moving? Make sure they're selected
